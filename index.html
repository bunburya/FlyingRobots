FlyingRobots by bunburya

A 3D clone of bsd-robots.

FlyingRobots, or flybots, is a 3D clone of the classic robots game. You are being chased by robots, and the only way to kill them is to make them crash. Whereas in the classic robots game you could move along the x and y axes, flybots allows you to also move along the z axis (ie perpendicular to the screen). It is written in Python 3 and currently uses Python's curses library, and is therefore restricted to Unix-like systems only. A cross-platform user interface is planned in the near future.

Dependencies

The only current dependency is python3, although the game currently only runs on Unix-like systems. The game should be able to run on Windows in the near future.

Install

Clone the git repository, or download and extract its contents. In the command line, cd to the directory containing the files and type "python3 setup.py install". The game can then be played by typing "flybots" in a terminal. If you use ArchLinux, you can also download and install the package from the AUR: http://aur.archlinux.org/packages.php?ID=57784
